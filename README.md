# 📚 Online Bookstore - Django Project

## 📖 Overview
This is a full-stack web application developed using Django that allows users to browse and purchase books online. The project follows the MVT (Model-View-Template) architecture and provides a smooth and user-friendly shopping experience.

## 🚀 Features
- 🔐 User Authentication (Login/Register/Logout)
- 📚 Book Listing and Categories
- 🔍 Search Functionality
- 🛒 Shopping Cart System
- 📦 Order Management
- 🧾 Checkout Process
- 🖥️ Responsive User Interface

## 🛠️ Tech Stack
- **Backend:** Python, Django
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Database:** SQLite
- **Tools:** Git, GitHub
